# Basic Econometrics (HS 163, IIT-Guwahati) Course Project:
1) This project consists of a multiple linear regression analysis of gross industrial output against variables like capital investment, labour etc.
2) A possible refinement in this project might be a better selection of relevant regressor variables; of course, not **all** variables from the raw dataset have been chosen as regressor variables:      definitions of terms from economics, _and_ correlation tests were used to narrow down the variables, but even then, a better selection is possible.
